# User Cases

## 1. Open the App
- User taps the button in the Telegram bot
- TrustGram Mini App opens

## 2. Start a Chat
- User enters the Telegram username of the recipient
- Session is established automatically (ECDH key exchange)
- Recipient receives a Telegram notification

## 3. Messaging
- User sends text messages
- Messages are encrypted in the browser before sending
- Recipient receives messages in real time

## 4. End a Chat
- User closes the session
- Recipient receives a Telegram notification that the session was closed
